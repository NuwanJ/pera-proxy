--------------------------------------------------------------------------------------------
Option 1
--------------------------------------------------------------------------------------------
VBS scripts

1. Click on 'proxyOn.vbs' file to turn on the proxy server, cachex.pdn.ac.lk in port 3128
2. Click on 'proxyOff.vbs' file to turn off the proxy configuration

--------------------------------------------------------------------------------------------
Option 2
--------------------------------------------------------------------------------------------
Proxy.pac file - Proxy Auto Configuration

1. Copy the proxy.pac file into your C:// partition
2. Goto proxy configuration window (Win10) and enable ‘use setup script’
3. Add following URL as ‘script address’.
	file://C:\proxy.pac
